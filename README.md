# 💰 Finance AI Advisor

An intelligent personal finance web application that helps users manage their money smarter using the power of **IBM Watson AI** and **IBM Natural Language Understanding**.

---

## 🌟 What is this project?

Finance AI Advisor is a full-stack web application that combines **Artificial Intelligence** with **personal finance management**. It uses IBM Cloud's AI services to provide smart financial guidance, analyze user goals through natural language, and help users track their spending habits — all in one beautiful dashboard.

---

## 🤖 IBM Cloud Services Used

| Service | Role in Project |
|--------|----------------|
| **IBM Watson Assistant** | Powers the intelligent finance chatbot that answers questions about budgeting, saving, investing, and debt management |
| **IBM Natural Language Understanding (NLU)** | Analyzes user-written financial goals and extracts keywords, sentiment, and key entities |
| **IBM Cloudant** | NoSQL cloud database used to store and retrieve user transactions |

---

## 💡 What Can You Do With This App?

- 💬 **Chat with an AI Finance Bot** — Ask anything about money and get instant smart advice powered by IBM Watson
- 🔍 **Analyze Your Financial Goals** — Type your financial concern and IBM NLU extracts key insights and sentiment
- ➕ **Track Your Expenses** — Add daily expenses by category (Food, Transport, Shopping, Bills, Health, etc.)
- 📊 **Visualize Spending** — See a beautiful pie chart of where your money is going
- 📋 **View Transaction History** — See your recent spending in a clean list
- 🧠 **Get Smart Suggestions** — The app gives personalized financial tips based on what you ask
